# ChronoRift Playable

Public static GitHub Pages runtime for ChronoRift playable builds.

## Security boundary

This repository is a disposable public layer for iPhone testing and safe game iteration.

- Static HTML/CSS/JS only.
- No secrets.
- No dependencies.
- No package manager.
- No GitHub Actions in the initial runtime path.
- No cross-repo access.
- No modification of ancestor/private repositories.

## Controls

- Joystick: move.
- Touch map: aim/select.
- Rift A/B: place linked rifts.
- Time: create a future echo.
- Lens: show temporal trace.
- Field: attract block.
- Clear: remove one echo.
- ⚠: SelfDebug report list.
